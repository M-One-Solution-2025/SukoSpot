# SukoSpot
SukoSpot lahir dari ide sederhana "gimana kalau kita bikin cara yang lebih mudah buat nemuin tempat seru di Sukoharjo?" kami juga ingin membantu memperkenalkan potensi kuliner lokal dan membuatnya lebih mudah dijangkau oleh siapa saja baik warga lokal maupun pengunjung dari luar daerah. Dari situ, Tim M-One Solution Software House mulai merancang aplikasi ini.
